# Piece artwork attribution

The base chess piece artwork in this directory is the "cburnett" set by
**Colin M.L. Burnett**, created for Wikipedia and used by Lichess, obtained
from the lichess-org/lila repository (`public/piece/cburnett/`).

Licensed under GFDL / BSD / GPL (triple license, per the original release).
The tinted per-seat variants in `../pieces/` are mechanically recolored
derivatives generated by `scripts/generate-pieces.mjs` and carry the same
license terms.
